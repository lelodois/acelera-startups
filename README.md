# Acelera Startups Backend

Versão backend que disponibiliza uma Api para cadastro de startup e empresas

### Pré-requisitos

```
  Java
  Mysql
  Maven
```

### Instalação

* Faça o clone do projeto
* mvn install
* java -jar target/acelera-startups-1.0.0.jar
  
### Tecnologias

* Java
* Mysql
* Spring framework + boot
* Spring data
* Swagger
